# Yoga_App_360

Yoga_App_360 is an Android application built using Java that provides a comprehensive yoga practice experience. 
It demonstrates Android development best practices with layered architecture, modern UI elements, and a focus on health and wellness.

---

## Features

- Guided yoga session library with various poses and sequences  
- Timer and progress tracking for sessions  
- Instructional visuals or animations for yoga poses  
- Modern UI built using Material Design components  

---

## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/skubaid5837/Yoga_App_360.git
```

### 2. Open in Android Studio
- Use Android Studio Ladybug (or newer)
- Import as a Gradle project

### 3. Build & Run
- Select an emulator or physical device
- Click Run in Android Studio
